# 🏢 MessMaintain: The Ultimate Mess Management SaaS

[![Next.js](https://img.shields.io/badge/Next.js-15-black?style=for-the-badge&logo=next.js)](https://nextjs.org/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind-4.0-38B2AC?style=for-the-badge&logo=tailwind-css)](https://tailwindcss.com/)
[![MongoDB](https://img.shields.io/badge/MongoDB-Latest-47A248?style=for-the-badge&logo=mongodb)](https://www.mongodb.com/)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg?style=for-the-badge)](LICENSE)

**MessMaintain** is a premium, feature-rich web application designed to eliminate the chaos of shared living. Whether you're in a student hostel or a bachelor mess, MessMaintain automates your accounting, meal tracking, and expense splitting with precision and style.

---

## ✨ Key Features

### 🍱 Advanced Meal Tracking
- **Multi-Meal Counter**: Support for guest meals (Breakfast, Lunch, Dinner).
- **Monthly Summaries**: Interactive grid view showing everyone's eating habits at a glance.
- **Dynamic Rates**: Real-time meal rate calculation based on total bazar costs.

### 💰 Professional Expense Management
- **Transaction Categories**: Dedicated tabs for Deposits, Bazar Costs, Shared Bills, and Individual Costs.
- **Sleek Modal System**: Quick-add transactions via a premium modal interface.
- **Smart Search & Pagination**: Effortlessly find any transaction among hundreds.

### 👥 Member Management
- **Role-Based Access**: Secure Manager/Member roles.
- **Join Requests**: Private messes with secure join request approvals.
- **Personal Dashboards**: Every member sees their own balance, deposits, and meal history.

### 🎨 State-of-the-art UI/UX
- **Dark & Light Mode**: Beautifully optimized for both themes.
- **Fully Responsive**: Flawless experience on desktops, tablets, and smartphones.
- **Modern Tech**: Built with Next.js 15, Tailwind 4, and Lucide Icons.

---

## 🛠 Tech Stack

- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **Database**: [MongoDB](https://www.mongodb.com/) with Mongoose
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Auth**: JWT-based secure authentication
- **Icons**: [Lucide React](https://lucide.dev/)
- **Charts**: [Recharts](https://recharts.org/)

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/mehedihasanfoysal8/mess_maintain_web_app.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up Environment Variables**:
   Create a `.env.local` file in the root:
   ```env
   MONGODB_URI=your_mongodb_connection_string
   JWT_SECRET=your_secure_random_string
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```

5. **Open the app**:
   Navigate to [http://localhost:3000](http://localhost:3000)
